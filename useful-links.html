Testing

The testing code is in the test and e2e-folder. There are three types of tests: server-unit-tests (inside test/server), client-unit-tests (inside test/client/unit) and end-to-end-tests (inside e2e). The unit-tests use the mocha-framework. For end-to-end-tests angular-scenario is used.
